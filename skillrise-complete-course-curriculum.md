# SkillRise — Complete Course Curriculum

This document contains module-by-module syllabi for all 50 courses across 10 categories. Each course includes duration, level, target earning outcome, and a complete week-by-week breakdown.

- **Week 3 — Applied Prompting:** Prompting for writing, coding, data analysis, and business use cases; building reusable prompt templates
